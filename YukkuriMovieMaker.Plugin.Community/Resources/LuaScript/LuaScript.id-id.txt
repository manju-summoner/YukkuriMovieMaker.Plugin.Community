- Tentang Efek Script Lua -

        Efek video "Script Lua" di YukkuriMovieMaker4 menggunakan
        bahasa Lua untuk penulisan skrip.
        Mesin skrip yang digunakan adalah MoonSharp (kompatibel dengan Lua 5.2).

        * Efek ini adalah implementasi mandiri dan berbeda dari spesifikasi skrip AviUtl2.
        * Memiliki spesifikasi yang berbeda dari file skrip AviUtl2 (*.anm2, dll.).
        * Karena IsAviUtlSupported = false, output EXO untuk AviUtl tidak didukung.


- Catatan -

        o Mesin eksekusi skrip adalah MoonSharp (kompatibel dengan Lua 5.2).
        o Pustaka standar yang tersedia terbatas pada berikut ini:
            table, string, math, bit32
            * io, os, debug, ffi, require, dll. tidak tersedia.
        o Batas waktu skrip adalah 5000ms (5 detik).
            Saat habis waktu, eksekusi dihentikan paksa dan diperlakukan sebagai kesalahan.
        o Skrip tidak dapat mengakses sistem file (termasuk require).
        o Pemanggilan skrip eksternal dari dalam skrip tidak didukung.
        o Operasi piksel (getpixel/setpixel/getpixeldata) mengakses gambar masukan aktual,
            sehingga transfer data GPU→CPU hanya terjadi pada akses pertama.
            Menghilangkan panggilan ini saat tidak diperlukan akan mempercepat pemrosesan.


- Parameter -

        Panel efek memiliki 4 trackbar berikut (mendukung animasi).
        Setiap nilai dapat direferensikan dalam skrip sebagai obj.track0 ~ obj.track3.

        Track0 : Parameter 0 (Default: 0, Rentang: Minimum ~ Maksimum)
        Track1 : Parameter 1 (Default: 0, Rentang: Minimum ~ Maksimum)
        Track2 : Parameter 2 (Default: 0, Rentang: Minimum ~ Maksimum)
        Track3 : Parameter 3 (Default: 0, Rentang: Minimum ~ Maksimum)

        * Setiap trackbar mendukung slider animasi (dengan interpolasi).
        * File .lua dapat diimpor/diekspor dari toolbar editor skrip.


- Variabel Global -

        Variabel-variabel berikut dapat direferensikan langsung dalam lingkup global skrip.

        time          : Waktu saat ini dalam detik relatif terhadap objek [Hanya Baca]
                        * Nilai yang sama dengan obj.time.
        frame         : Nomor frame saat ini relatif terhadap objek [Hanya Baca]
                        * Nilai yang sama dengan obj.frame.
        totalframe    : Total jumlah frame objek [Hanya Baca]
                        * Nilai yang sama dengan obj.totalframe.
        framerate     : Kecepatan frame (fps) [Hanya Baca]
                        * Nilai yang sama dengan obj.framerate.
        timelineframe : Nomor frame saat ini relatif terhadap seluruh timeline [Hanya Baca]
                        * obj.frame adalah frame relatif dari awal objek,
                          sedangkan ini adalah frame absolut dari awal timeline.
        timelinetime  : Waktu saat ini dalam detik relatif terhadap seluruh timeline [Hanya Baca]
        layer         : Nomor layer tempat objek ditempatkan [Hanya Baca]
                        * Nilai yang sama dengan obj.layer.
        obj           : Tabel informasi objek (lihat «Tabel obj» di bawah)
        scene         : Tabel informasi adegan (lihat «Tabel scene» di bawah)
        ymm4          : YMM4 specific meta-information table (see "ymm4 Table" below)
        anim          : Animation utility table (see "anim Table" below)


- Tabel scene -

        scene.width   : Resolusi horizontal (piksel) adegan (proyek) [Hanya Baca]
        scene.height  : Resolusi vertikal (piksel) adegan (proyek) [Hanya Baca]


- Tabel obj - Variabel -

        Variabel-variabel berikut menyimpan informasi render objek saat ini.
        * Variabel yang tidak ditandai Hanya Baca dapat dimodifikasi dalam skrip, dan perubahannya akan tercermin dalam render.

        o Koordinat / Offset
        obj.x         : Koordinat referensi tampilan X [Hanya Baca]
                        * Setara dengan «Koordinat Referensi Tampilan X» dalam spesifikasi AviUtl2.
                        * Perubahan yang dilakukan di dalam skrip tidak dibaca kembali dan tidak berpengaruh.
                        * Ditetapkan secara internal dari DrawDescription.Draw.X;
                          perubahan berlaku setelah skrip dijalankan.
        obj.y         : Koordinat referensi tampilan Y [setara Hanya Baca]
                        * Ditetapkan secara internal dari DrawDescription.Draw.Y.
        obj.z         : Koordinat referensi tampilan Z [setara Hanya Baca]
                        * Ditetapkan secara internal dari DrawDescription.Draw.Z.
        obj.ox        : Offset pusat X (dapat dimodifikasi)
                        * Ditetapkan secara internal dari DrawDescription.CenterPoint.X (inDesc).
                        * Setara dengan «Koordinat X relatif dari referensi (obj.ox)» di AviUtl2,
                          tetapi dalam plugin ini diperlakukan sebagai CenterPoint (offset titik pusat).
        obj.oy        : Offset pusat Y (dapat dimodifikasi)
                        * Ditetapkan secara internal dari DrawDescription.CenterPoint.Y (inDesc).

        o Skala / Transformasi
        obj.zoom      : Faktor skala (dapat dimodifikasi)
                        * 1.0 adalah skala normal (bukan 0.0), kompatibel dengan AviUtl2.
                        * Dihitung secara internal sebagai rata-rata ZoomX dan ZoomY (zoomAvg).
                        * Saat output, diperluas ke zoom X/Y individual sebagai obj.zoom * (1 ± obj.aspect).
        obj.aspect    : Rasio aspek (dapat dimodifikasi), Rentang: -1.0 ~ 1.0
                        * Nilai positif mengecilkan arah X; nilai negatif mengecilkan arah Y.
                        * Dihitung secara internal sebagai (ZoomX - ZoomY) / (ZoomX + ZoomY).
        obj.sx        : Scale factor in X direction (writable) 1.0 = 100%
                        * Precise independent scale control specific to YMM4.
                        * Takes precedence over zoom/aspect modifications.
        obj.sy        : Scale factor in Y direction (writable) 1.0 = 100%
                        * Precise independent scale control specific to YMM4.
                        * Takes precedence over zoom/aspect modifications.
        obj.alpha     : Opasitas (dapat dimodifikasi), Rentang: 0.0 ~ 255.0
                        * AviUtl2 menggunakan 0.0~1.0, tetapi plugin ini menggunakan 0.0~255.0.
                        * Ditetapkan secara internal dari DrawDescription.Opacity * 255.
                        * Dibagi 255 dan dinormalisasi ke 0.0~1.0 pada output (dengan Clamp).
                        * Skrip default menetapkan obj.alpha = math.min(time * 255, 255),
                          memberikan implementasi fade-in default.

        o Rotasi
        obj.rx        : Sudut rotasi sumbu X (dapat dimodifikasi), Satuan: derajat (°), 360.0 = satu putaran penuh
                        * Ditetapkan secara internal dari DrawDescription.Rotation.X.
        obj.ry        : Sudut rotasi sumbu Y (dapat dimodifikasi), Satuan: derajat (°), 360.0 = satu putaran penuh
                        * Ditetapkan secara internal dari DrawDescription.Rotation.Y.
        obj.rz        : Sudut rotasi sumbu Z (dapat dimodifikasi), Satuan: derajat (°), 360.0 = satu putaran penuh
                        * Ditetapkan secara internal dari DrawDescription.Rotation.Z.
        obj.rxr       : Rotation angle around X axis in radians (writable)
                        * Takes precedence if rx is also modified simultaneously.
        obj.ryr       : Rotation angle around Y axis in radians (writable)
                        * Takes precedence if ry is also modified simultaneously.
        obj.rzr       : Rotation angle around Z axis in radians (writable)
                        * Takes precedence if rz is also modified simultaneously.

        o Ukuran Gambar
        obj.w         : Lebar gambar masukan dalam piksel [Hanya Baca]
                        * Secara internal adalah bilangan bulat yang diturunkan dari D2D GetImageLocalBounds.
        obj.h         : Tinggi gambar masukan dalam piksel [Hanya Baca]
                        * Secara internal adalah bilangan bulat yang diturunkan dari D2D GetImageLocalBounds.
        obj.hw        : Half of the input image width (= obj.w / 2) [ReadOnly]
        obj.hh        : Half of the input image height (= obj.h / 2) [ReadOnly]
        obj.diagonal  : Panjang diagonal dari gambar input (= math.sqrt(w*w + h*h)) [ReadOnly]
        obj.cx        : Koordinat X pusat gambar (= obj.w / 2) [Hanya Baca]
                        * Koordinat pusat referensi untuk operasi piksel.
        obj.cy        : Koordinat Y pusat gambar (= obj.h / 2) [Hanya Baca]

        o Waktu / Frame
        obj.time      : Waktu saat ini dalam detik relatif terhadap objek [Hanya Baca]
                        * Nilai yang sama dengan variabel global time.
                        * Ditetapkan secara internal dari EffectDescription.ItemPosition.Time.TotalSeconds.
        obj.frame     : Nomor frame saat ini relatif terhadap objek [Hanya Baca]
                        * Nilai yang sama dengan variabel global frame.
                        * Ditetapkan secara internal dari EffectDescription.ItemPosition.Frame.
        obj.totalframe: Total jumlah frame objek [Hanya Baca]
                        * Nilai yang sama dengan variabel global totalframe.
                        * Ditetapkan secara internal dari EffectDescription.ItemDuration.Frame.
        obj.framerate : Kecepatan frame (fps) [Hanya Baca]
                        * Nilai yang sama dengan variabel global framerate.
                        * Ditetapkan secara internal dari EffectDescription.FPS.

        o Trackbar
        obj.track0    : Nilai saat ini Track0 di panel efek [Hanya Baca]
                        * Nilai setelah interpolasi animasi.
        obj.track1    : Nilai saat ini Track1 di panel efek [Hanya Baca]
        obj.track2    : Nilai saat ini Track2 di panel efek [Hanya Baca]
        obj.track3    : Nilai saat ini Track3 di panel efek [Hanya Baca]

        o Layer
        obj.layer     : Nomor layer tempat objek ditempatkan [Hanya Baca]
                        * Nilai yang sama dengan variabel global layer.
                        * Ditetapkan secara internal dari EffectDescription.Layer.


- Tabel obj - Perubahan Variabel dan Pencerminan ke Render -

        Setelah eksekusi skrip, hanya variabel-variabel berikut yang dibaca kembali ke informasi render (diterapkan).

        Variabel yang diterapkan:
            obj.x, obj.y, obj.z
            obj.ox, obj.oy
            obj.zoom, obj.aspect
            obj.alpha
            obj.rx, obj.ry, obj.rz

        * obj.w, obj.h, obj.cx, obj.cy, obj.frame, obj.time,
          obj.totalframe, obj.framerate, obj.track0 ~ obj.track3,
          obj.layer adalah Hanya Baca dan perubahannya tidak akan tercermin dalam render.

        Rumus nilai output:
            ZoomX = obj.zoom * (1.0 + obj.aspect)
            ZoomY = obj.zoom * (1.0 - obj.aspect)
            Opacity = Clamp(obj.alpha / 255.0, 0.0, 1.0)
            Draw = (obj.x, obj.y, obj.z)
            CenterPoint = (obj.ox, obj.oy)
            Rotation = (obj.rx, obj.ry, obj.rz)


- Tabel obj - Fungsi -

        o obj.getpixel(x, y)
                Mengambil informasi piksel pada koordinat yang ditentukan.
                * Transfer data gambar GPU→CPU terjadi pada panggilan pertama (dengan overhead pemrosesan).
                * Koordinat piksel menggunakan sudut kiri atas gambar sebagai (0, 0), dengan kanan dan bawah sebagai positif.
                * Nilai yang dikembalikan adalah ekuivalen alpha langsung yang dikonversi dari alpha premultiplied.
                x       : Koordinat X (0 ~ obj.w - 1)
                y       : Koordinat Y (0 ~ obj.h - 1)
                Mengembalikan : 4 nilai r, g, b, a (nilai kembalian jamak)
                          r = Komponen merah (0.0 ~ 255.0)
                          g = Komponen hijau (0.0 ~ 255.0)
                          b = Komponen biru (0.0 ~ 255.0)
                          a = Nilai alpha (0.0 ~ 255.0)
                          * Mengembalikan (0, 0, 0, 0) untuk koordinat di luar rentang.
                Contoh: r, g, b, a = obj.getpixel(10, 20)

        o obj.setpixel(x, y, r, g, b [, a])
                Menimpa informasi piksel pada koordinat yang ditentukan.
                * Data piksel secara otomatis ditandai sebagai «dimodifikasi» saat dipanggil,
                  dan gambar yang dimodifikasi dikeluarkan setelah skrip berakhir.
                * Disimpan secara internal dalam format alpha premultiplied (BGRA).
                  Konversi premultiplied dilakukan saat setpixel.
                x       : Koordinat X (0 ~ obj.w - 1)
                y       : Koordinat Y (0 ~ obj.h - 1)
                r       : Komponen merah (0.0 ~ 255.0)
                g       : Komponen hijau (0.0 ~ 255.0)
                b       : Komponen biru (0.0 ~ 255.0)
                a       : Nilai alpha (default: 255.0)
                Contoh: obj.setpixel(10, 20, 255, 0, 0, 128)

        o obj.getpixeldata()
                Mengembalikan objek proxy untuk akses massal ke semua data piksel.
                * Manipulasi piksel lebih cepat dibandingkan getpixel/setpixel.
                * Transfer data gambar GPU→CPU terjadi pada panggilan pertama (dengan overhead pemrosesan).
                Mengembalikan : Objek PixelDataProxy
                          .width  : Lebar gambar dalam piksel (= obj.w)
                          .height : Tinggi gambar dalam piksel (= obj.h)
                          :get(index)        : Mendapatkan nilai saluran berdasarkan indeks
                          :set(index, value) : Menetapkan nilai saluran berdasarkan indeks

                Spesifikasi indeks:
                    Indeks dimulai dari 1 (mengikuti konvensi Lua).
                    Urutan saluran adalah R, G, B, A berulang.
                    Indeks saluran R untuk pixel(x, y) =
                        (y * obj.w + x) * 4 + 1
                    Indeks saluran G untuk pixel(x, y) =
                        (y * obj.w + x) * 4 + 2
                    Indeks saluran B untuk pixel(x, y) =
                        (y * obj.w + x) * 4 + 3
                    Indeks saluran A untuk pixel(x, y) =
                        (y * obj.w + x) * 4 + 4

                    * Memodifikasi alpha (saluran 4) juga akan mengkonversi RGB secara proporsional.
                    * Memodifikasi saluran RGB akan menerapkan konversi premultiplied.
                    * Nilai yang diambil adalah ekuivalen alpha langsung, sama seperti get(index).

                Contoh:
                    local pd = obj.getpixeldata()
                    local idx = (10 * obj.w + 20) * 4 + 1  -- Saluran R dari (x=20, y=10)
                    local r = pd:get(idx)
                    pd:set(idx, 128)

        o obj.putpixeldata()
                Fungsi kosong (no-op) yang didefinisikan untuk kompatibilitas.
                * Dalam skrip AviUtl, getpixeldata/putpixeldata digunakan bersama,
                  tetapi dalam plugin ini perubahan melalui setpixel/set diterapkan secara otomatis,
                  sehingga tidak perlu memanggil fungsi ini.



- ymm4 Table -

        Table holding YMM4-specific meta-information. All values are ReadOnly.

        ymm4.group_index        : Current item index within a group control (0-based)
        ymm4.group_count        : Total number of items within a group control
        ymm4.group_ratio        : Progress ratio within the group (group_index / group_count)
        ymm4.timeline_totalframe: Total number of frames for the entire timeline
        ymm4.timeline_totaltime : Total time in seconds for the entire timeline
        ymm4.time_ratio         : Current playback position ratio (0.0 ~ 1.0)
        ymm4.is_saving          : Whether video is currently being exported (true only during export)
        ymm4.is_playing         : Whether the timeline is currently playing (true only during playback)
        ymm4.is_paused          : Whether the timeline is currently paused (true only when paused)
        ymm4.scene_id           : Unique ID (Guid string) of the current scene


- anim Table -

        Pure utility functions and constants useful for animation calculations.

        anim.tau                : 2 * Pi (math.pi * 2)
        anim.e                  : Bilangan Euler (math.exp(1))
        anim.phi                : Rasio emas ((1 + math.sqrt(5)) / 2)
        anim.sqrt2              : Akar kuadrat dari 2 (math.sqrt(2))
        anim.lerp(a, b, t)      : Linear interpolation between a and b by t (0.0 ~ 1.0)
        anim.smoothstep(e0, e1, x): Smooth Hermite interpolation between e0 and e1
        anim.smootherstep(e0, e1, x): Interpolasi Hermite yang lebih halus antara e0 dan e1 (turunan kedua kontinu)
        anim.clamp(v, lo, hi)   : Clamps value v between lo and hi
        anim.map(v, a1, b1, a2, b2): Maps value v from range a1~b1 to range a2~b2
        anim.norm(v, lo, hi)    : Gets the normalized ratio (0.0 ~ 1.0) of v within lo~hi
        anim.wrap(v, lo, hi)    : Wraps value v to loop within the range lo~hi
        anim.pingpong(t, len)   : Value that bounces back and forth between 0 and len
        anim.sign(v)            : Gets the sign of v (1 for positive, -1 for negative, 0 for zero)
        anim.oscillate(t, lo, hi, freq): Smoothly oscillates between lo and hi at the given freq (sine wave)
        anim.triangle(t, freq)  : Triangle wave oscillating between 0.0 and 1.0 at the given freq
        anim.square(t, freq)    : Square wave switching between 1.0 and 0.0 at the given freq
        anim.duration(t, dur)   : Current ratio over duration dur (0.0 ~ 1.0, clamped)
        anim.delay(t, d)        : Time starting after delay d (0 if negative)
        anim.ease_in(t)         : Quadratic ease-in (acceleration)
        anim.ease_out(t)        : Quadratic ease-out (deceleration)
        anim.ease_in_out(t)     : Ease-in-out kuadratik (akselerasi kemudian deselerasi)
        anim.elastic(t)         : Bouncy spring easing
        anim.back(t)            : Easing that slightly overshoots before returning
        anim.step(edge, x)      : Returns 1.0 if x >= edge, otherwise 0.0
        anim.fract(v)           : Gets only the fractional part of v
        anim.bounce(t)          : Bouncing ease effect
        anim.hsv_to_rgb(h, s, v): Converts HSV (Hue 0~360, Saturation 0~1, Value 0~1) to RGB (0~255)
        anim.rgb_to_hsv(r, g, b): Converts RGB (0~255) to HSV
        anim.noise(x, y, z)     : 1D to 3D continuous smooth noise (-1.0 ~ 1.0)
        anim.rand(min, max, s)  : Stable random value based on seed `s`
        anim.len(x, y)          : Mendapatkan panjang (norma) dari vektor (x, y)
        anim.dist(x1, y1, x2, y2): Mendapatkan jarak antara dua titik (x1, y1) dan (x2, y2)
        anim.dot(x1, y1, x2, y2): Mendapatkan hasil kali titik (dot product) dari dua vektor (x1, y1) dan (x2, y2)
        anim.normalize(x, y)    : Menormalisasi vektor (x, y) mengembalikan vektor arah (Tuple)
        anim.polar(r, a)        : Converts polar coordinates (radius, angle) to (x, y) tuple
        anim.rotate(x, y, a)    : Rotates 2D point (x, y) by angle `a` (degrees) returning (x, y) tuple
        anim.bezier(t, p0, p1, p2, p3): Evaluates cubic bezier curve at `t` (0.0~1.0)

- Contoh Skrip -

        o Fade in (skrip default)
                obj.alpha = math.min(time * 255, 255)

        o Fade out
                obj.alpha = math.max((obj.totalframe - obj.frame) / obj.framerate * 255, 0)

        o Mengontrol opasitas dengan Track0
                obj.alpha = obj.track0

        o Rotasi seiring waktu
                obj.rz = time * 90  -- 90 derajat per detik

        o Membalik komponen merah piksel
                for y = 0, obj.h - 1 do
                    for x = 0, obj.w - 1 do
                        local r, g, b, a = obj.getpixel(x, y)
                        obj.setpixel(x, y, 255 - r, g, b, a)
                    end
                end

        o Konversi skala abu-abu menggunakan getpixeldata (cepat)
                local pd = obj.getpixeldata()
                local w = pd.width
                local h = pd.height
                for y = 0, h - 1 do
                    for x = 0, w - 1 do
                        local base = (y * w + x) * 4
                        local r = pd:get(base + 1)
                        local g = pd:get(base + 2)
                        local b = pd:get(base + 3)
                        local a = pd:get(base + 4)
                        local gray = r * 0.299 + g * 0.587 + b * 0.114
                        pd:set(base + 1, gray)
                        pd:set(base + 2, gray)
                        pd:set(base + 3, gray)
                    end
                end

        o Mengontrol zoom dengan Track0 (1.0 = skala normal)
                obj.zoom = obj.zoom * (1 + obj.track0 / 100)
